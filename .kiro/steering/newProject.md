# Crear Proyecto Nuevo

Cuando se cargue este steering, leer y ejecutar en orden todos los archivos dentro de la carpeta `newProject/`.

Los archivos están numerados y deben ejecutarse en secuencia:

1. #[[file:.kiro/steering/newProject/1.projectStructure.md]] — Crear la estructura de carpetas del proyecto
2. #[[file:.kiro/steering/newProject/2.createBaseFiles.md]] — Crear los archivos base estándar
3. #[[file:.kiro/steering/newProject/3.createCssFiles.md]] — Crear los archivos CSS
4. #[[file:.kiro/steering/newProject/4.runProject.md]] — Inicializar proyecto e instalar dependencias

## Instrucciones

- Ejecutar cada paso en orden numérico
- Respetar siempre las convenciones de `namingConventions.md` (cargado automáticamente)
- No saltear pasos
- Confirmar con el usuario cuando se complete todo
