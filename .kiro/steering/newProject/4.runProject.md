# Inicializar Proyecto

Ejecutar los siguientes comandos en la terminal para inicializar el proyecto e instalar las dependencias.

## Comandos a ejecutar

1. `npm init -y`
2. `npm install express ejs method-override multer express-validator express-session bcryptjs node-fetch@2 exceljs pdfkit pdf-lib archiver sequelize-cli sequelize mysql2`
