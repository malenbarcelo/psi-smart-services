# Crear Archivos CSS

Crear los archivos CSS estándar del proyecto. Solo crear los archivos listados acá con el contenido especificado. No agregar nada extra.

## Archivos a crear

### public/css/generalStyles.css

```css
/* colors */
:root {
  --color1: red;
  --color2: red;
  --color3: red;
  --color4: red;
  --color5: red;
  --color6: red;
  --errorColor: rgb(221, 7, 7);
  --errorColorLight: rgb(202, 127, 127);
  --okColor: green;
  --okColorLight: rgb(173, 204, 173);
}

/* body */
body {
  font-family: 'Poppins', Arial, "sans-serif";
  margin: 0;
  font-size: 14px;
  color: #333;
  letter-spacing: 0.3px;
}

/* inputs */
input, select, textarea {
  outline: none;
}

input::-webkit-outer-spin-button,
input::-webkit-inner-spin-button {
  -webkit-appearance: none;
  margin: 0;
}

/* Hide arrows from input type number */
input[type="checkbox"]:checked {
  accent-color: black;
}

input[type="radio"]:checked {
  accent-color: black;
}

a {
  text-decoration: none;
  color: black;
}
```
